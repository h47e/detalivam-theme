# Волна 1: Что делать этой неделей (практически)

## Стартовый приоритет (High)
1. Назначить целевые URL для запросов 1–12 из `TRAFFIC_SEMANTICS_FIRST_BATCH.md` (крылья/пороги/арки/бамперы по моделям).
2. На каждой выбранной посадочной:
   - проверить наличие блока "Наличие" и "Цена";
   - добавить `SEO Title` с форматом `[{деталь}] [{модель}] купить в Тольятти | ДеталиВам`;
   - добавить `SEO Description` с упором на цену/наличие/самовывоз.
3. Для запросов с артикулом (18–26) убедиться, что они попадают либо на карточку, либо на фильтр-подкатегорию с привязкой к SKU.
4. Убедиться, что на этих посадках есть CTA на:
   - добавление в корзину (основная конверсия);
   - клик в Ozon (для альтернативного канала).

## Средний приоритет (Medium)
5. Вынести первые 10 landing/FAQ в контент:
   - 37, 38, 39, 40, 41 из `TRAFFIC_SEMANTICS_FIRST_BATCH.md`.
6. Уточнить внутренние ссылки:
   - блок "С этим товаром смотрят также";
   - блок "Подбор по модели/артикулу";
   - блок FAQ по каждому посадочному кластера.
7. Подготовить черновой список редиректов/каноникал для дублей запросов (например, пробелы/дефисы в SKU, безпробельные варианты «крыловаз»).

## Низкий приоритет (Low)
8. Закрыть контентную таблицу (если есть ресурсы) для 30–50 запросов в этом же формате:
   - добавить финальные URL;
   - добавить ожидаемую конверсию и целевой канал.
9. Проверить поведение в выдаче:
   - CTR на выдаче для новых посадок;
   - конверсию во внутренний поиск;
   - количество заходов в Ozon.

## Формат для правок в админке
- Для каждой строки обязательно заполняем:
  - `целевая посадка`,
  - `главная цель Метрики`,
  - `вторичная цель`,
  - `ответственный`,
  - `срок`.

Рабочая таблица ведётся в `TRAFFIC_SEMANTICS_BATCH1_WORKBOARD.md`.
Сначала закрываем строки со сроком `Неделя 1`, затем `Неделя 2`, затем контентные landing-страницы.

## Мини-метрики успеха волны 1
- 80%+ заявок по этим посадкам должны иметь заполненный Title/Description;
- не менее 15% посадок закрыты в "готово к индексации" (без технических дыр);
- по каждому блоку видно в Метрике хотя бы 1 из целевых действий в течение 2 недель.
