# ДеталиВам

WordPress/WooCommerce-тема для интернет-магазина автозапчастей «ДеталиВам».

## Описание

Тема собирает витрину магазина автозапчастей: каталог, карточки товаров, корзину, checkout, личный кабинет, сервисные страницы, SEO-инструменты и настройки контента из админки.

Основной акцент темы — удобная продажа кузовных деталей и элементов системы выпуска с понятным оформлением заказа, отложенной оплатой после проверки менеджером и управляемыми страницами для покупателей.

## Что входит

- адаптированная WooCommerce-витрина;
- страницы каталога, товара, корзины, checkout и личного кабинета;
- сервисные страницы «О компании», «Доставка», «Контакты», «Возврат»;
- настройки контента, профиля магазина и визуальных параметров из админки;
- SEO-панель, sitemap и диагностические инструменты;
- интеграция с избранным, сравнением, быстрым поиском и AJAX-корзиной;
- отдельные стили для order-pay, thank you и просмотра заказа.

## Структура

- `assets/css/main.css` — основные стили витрины;
- `assets/js/main.js` — фронтовая логика;
- `inc/` — модули темы, WooCommerce, SEO, админка и настройки;
- `woocommerce/` — переопределения шаблонов WooCommerce;
- `page-*.php` — шаблоны сервисных страниц.

## Требования

- WordPress 6.0 или выше;
- WooCommerce;
- PHP 8.0 или выше.

## Установка

1. Скопируйте папку темы в `wp-content/themes/clear`.
2. Активируйте тему «ДеталиВам» в админке WordPress.
3. Проверьте настройки в разделах `ДеталиВам -> Настройки`, `Магазин`, `Контент` и `SEO`.

## Репозиторий

В репозитории хранится только тема. Рабочие отчёты, история переписки, архивы и отдельные плагины не входят в этот репозиторий.
